# Stock-Trade-Simulator

Developed a stock trading simulator that models investment strategies over historical market data. Integrated live data retrieval with the yfinance API, along with data refactoring using the pandas/numpy Python libraries. Designed an interactive interface using Streamlit for parameter input and the plotly library for data visualization

I was doing development on my own over the summer of 2025 and never made a GitHub repo for some reason, But I assure you that this project was not vibe-coded, so ignore the 1 commit; this project has gone through many iterations

command to install dependencies: pip install streamlit yfinance pandas numpy matplotlib plotly
command to run: streamlit run app.py
